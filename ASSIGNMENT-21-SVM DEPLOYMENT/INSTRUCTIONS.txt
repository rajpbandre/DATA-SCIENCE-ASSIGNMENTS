   1) RUN python code [svm.ipynb] in jupyter notebook with pima dataset.
   
   2) THEN once this code is run, clf.pkl file will be created automatically & it will be saved in current working                                    directory. This clf.pkl file only for deployment purpose.

   4) THEN by updating the location of where all these files are present in anaconda promt.
